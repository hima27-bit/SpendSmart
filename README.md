# 💰 SpendSmart — Expense Data Interpreter & AI Financial Advisor

> A Python-based personal finance analytics web application that tracks your spending, visualizes patterns, and delivers AI-powered financial advice tailored to your actual data.

---

## 📌 Overview

SpendSmart is a full-stack data analytics project built with Python and Streamlit. It allows users to log daily expenses, analyze spending behavior through interactive charts and dashboards, and receive personalized financial recommendations generated by an AI model via the Groq API.

This project was built to demonstrate practical skills in data engineering, analytics, visualization, and AI API integration — applied to a real-world personal finance use case.

---

## ✨ Features

| Feature | Description |
|---|---|
| 📥 Expense Logging | Add expenses by date, category, amount, and description via sidebar form |
| 📊 KPI Dashboard | Real-time cards showing total spend, monthly total, average per day, and top category |
| 📈 Spending by Category | Horizontal bar chart sorted by highest spend |
| 📉 Monthly Trend | Line chart showing spending pattern across months |
| 🥧 Expense Share | Pie chart showing each category's percentage of total spend |
| 🧾 Recent Transactions | Table of the 10 most recent entries with delete functionality |
| 🤖 AI Financial Advisor | Groq-powered advice — savings goals, spending concerns, and quick wins |
| 🗂️ Data Persistence | All expenses saved locally to `expenses.csv` |
| 🔄 Sample Data | One-click generation of 3 months of realistic demo data |

## ⚙️ Setup and Installation

### 1. Clone the repository
```bash
git clone https://github.com/yourusername/spendsmart.git
cd spendsmart
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Set up your API key

Copy the example env file:
```bash
cp .env.example .env
```

Open `.env` and paste your Groq API key:
```
GROQ_API_KEY=gsk_your-actual-key-here
```

Get a free key at **console.groq.com** — no billing required, 14,400 requests/day free.

### 4. Run the app
```bash
streamlit run app.py
```

The app opens at `http://localhost:8501`

## 🚀 How to Use

1. **Add expenses** from the sidebar — select a date, category, enter amount and description
2. **Or load sample data** — click "Load Sample Data" for instant 3-month demo data
3. **View your dashboard** — KPI cards and charts update automatically
4. **Get AI advice** — click "Analyse My Spending & Get Advice" at the bottom
5. **Delete entries** — enter the row index in the transactions table and click Delete

### Expense Categories
`Food & Dining` · `Transport` · `Shopping` · `Entertainment` · `Utilities` · `Healthcare` · `Education` · `Others`
